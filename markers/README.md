# Marker Images

Place your marker pattern files (`.patt`) in this directory.

## Getting Started

1. **Print the Hiro Marker:**
   - Download the Hiro marker from: https://jeromeetienne.github.io/AR.js/data/images/HIRO.jpg
   - Print it on a white paper (at least 8x8 cm for best results)

2. **Create Custom Markers:**
   - Visit: https://jeromeetienne.github.io/AR.js/three.js/examples/marker-training/examples/generator.html
   - Upload your logo/image
   - Download the generated `.patt` file
   - Place it in this directory
   - Update the marker URL in `index.html`

## Tips for Best Results

- Use high contrast images
- Ensure good lighting
- Keep the marker flat and steady
- Print markers at least 8x8 cm (3x3 inches)
- Avoid reflective surfaces
